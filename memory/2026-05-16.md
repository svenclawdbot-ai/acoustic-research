**Topic: Lock-in Amplifier Signal Processing for Impedance Spectroscopy**
*Building on Friday night's soil probe work and acoustic NDE research*

**Objective:** Implement a digital lock-in amplifier on the ESP32-S3 that can:
1. Generate a swept-frequency sine excitation via AD9833 (1 kHz → 1 MHz)
2. Sample the TIA output at the excitation frequency (coherent sampling)
3. Extract magnitude and phase using quadrature demodulation (I/Q mixing)
4. Compute complex impedance Z(ω) = V_exc / I_soil across the frequency sweep
5. Fit a Cole-Cole model to extract soil moisture

**Deliverable:** Arduino sketch that runs the full measurement chain and prints |Z| and phase for each frequency point.

**Key concepts:**
- Coherent sampling (N samples per cycle, phase-locked)
- Digital mixing: multiply ADC samples by sin(ωt) and cos(ωt)
- Low-pass filtering via boxcar average over N cycles
- Phase unwrapping for accurate phase response
