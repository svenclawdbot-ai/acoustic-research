### FPGA Architecture
- Zynq PS + HP0 AXI + AXI Interconnect
- ADC/DAC interfaces (Pavel Demin's cores)
- AXI GPIO for DG408 MUX control (S0,S1,S2,/EN)
- AXI DMA S2MM for ADC data transfer
- BRAM for DAC waveform storage
- MUX controller state machine with auto-scan
