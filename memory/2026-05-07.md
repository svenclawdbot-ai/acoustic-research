### Soil Impedance Spectroscopy DSP Validation
**Part 1: Soil Impedance Model** ✓
- Wenner array geometry (a=5cm, K_g=0.314m)
- Three-state model: dry clay / moderate loam / saturated loam
- R_bulk + CPE (electrode polarization) model
- Key insight: usable moisture measurement at 100kHz–1MHz where bulk dominates
- File: `soil_impedance_model.py` + `soil_impedance_spectra.png`

**Part 2: Lock-in Amplifier DSP** ✓
- Full measurement chain: excitation → soil → TIA → ADC → lock-in → |Z|/phase
- Synthetic data testing with realistic noise
- STM32 (1MSPS, 12-bit) sufficient — Red Pitaya overkill
- Measurement time budget: ~44s for 50-point sweep
- File: `lockin_spectrometer.py` + `lockin_impedance_sweep.png`

**Part 3: Noise Budget** ✓
- ADC quantization dominates at all frequencies
- TIA gain optimization: 1kΩ–5kΩ optimal for 12-bit ADC
- ENOB ~10–12 bits across range, SNR 57–114 dB
- 12-bit STM32 sufficient for 0.5–1% accuracy
- File: `noise_budget_analysis.py` + `noise_budget_analysis.png`
