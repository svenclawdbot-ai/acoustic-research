## 💡 Findings & Insights

### Wave Physics
- **Kelvin-Voigt:** Excessive high-frequency damping (α ∝ ω²)
  - Challenging for phase extraction due to rapid attenuation
  - Good for inverse problem validation (simple parameterization)

- **Zener (Standard Linear Solid):** Moderate damping (α ∝ ω)
  - Bounded high-frequency response (G_∞)
  - 3× longer propagation distance than KV
  - Better physical model for soft tissue

### Multi-Layer Media
- Interface conditions: displacement and stress continuity
- Reflection coefficient: R = (Z₂ − Z₁)/(Z₂ + Z₁)
- Dispersion becomes implicit in layered media
- Guided modes possible in thin layers

### Technical Implementation
- Time-step constraints:
  - Wave CFL: dt ≤ 0.3·dx/c_s
  - Viscous: dt ≤ 0.5·ρ·dx²/(4η)
- PML with 20-cell thickness effective for open boundaries
- Minimum viscosity η ≥ 0.5 Pa·s required for stability
