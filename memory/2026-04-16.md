## GCC-PHAT Integration for Shear Wave Pipeline

Worked on integrating GCC-PHAT (Generalized Cross-Correlation with Phase Transform) into the shear wave dispersion extraction pipeline.

### Key Challenge: Phase Ambiguity
GCC-PHAT provides sub-sample precision delay estimation, but suffers from phase ambiguity when the delay is larger than the signal period. For a 100 Hz signal (10 ms period), a 7 ms delay looks identical to -3 ms or +17 ms in terms of phase.

### Solution: Two-Stage Approach
1. **Coarse delay**: Envelope peak detection to resolve which period
2. **Fine delay**: GCC-PHAT for sub-sample precision within that period

### Key Findings
1. GCC-PHAT requires narrowband signals - fails on overlapping broadband signals
2. Envelope detection is more robust for dispersive signals with overlapping frequency components
3. Frequency-domain phase extraction works best with isolated frequency bins
4. 2D FDTD simulation has numerical stability issues with soft tissue parameters (G'=2000 Pa, low viscosity)

### Fundamental Insight
For reliable dispersion extraction:
- Use sequential narrowband simulations (one frequency at a time), OR
- Use well-separated tone bursts in synthetic data where each frequency arrives at a different time
