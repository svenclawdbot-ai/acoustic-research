### 🎯 Research Question Selected

**Focus:** Shear wave dispersion in viscoelastic tissue
- Can we reliably separate elastic vs viscous components?
- Critical for disease characterization (cancer detection)
- Bridges NDE and medical imaging

### 🧮 Computational Model Built

Created `shear_wave_1d_simulator.py`:
- 1D FDTD simulation of shear wave propagation
- Kelvin-Voigt viscoelastic model implemented
- Absorbing boundary conditions (Mur 1st order)
- Three simulation modes:
  1. Single frequency propagation
  2. Elastic vs viscoelastic comparison
  3. Dispersion analysis demo

**Key Physics Captured:**
```
∂²u/∂t² = (G'/ρ) ∂²u/∂x² + (η/ρ) ∂³u/∂x²∂t
```
