## CRITICAL NOTES

1. **HV Supply (×2):** Order TWO. These fail often. £12 insurance policy.

2. **Gate Zener:** MUST add BZX55-C15 — without it, MOSFET dies from overvoltage.

3. **TVS Diode:** MUST add P6KE200A — piezo inductive kickback destroys unprotected circuits.

4. **ESP32-S3:** Buy DevKitC with built-in antenna — easier than adding external antenna.

5. **ADXL355:** Buy the module (not bare chip) — has voltage regulator and level shifters.

6. **Shipping:** Pay for expedited shipping on Priority 1 — 2 weeks is already long.
