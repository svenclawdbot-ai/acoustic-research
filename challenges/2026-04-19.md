## Guided Wave Defect Detection: Frequency-Wavenumber Analysis

### Problem Context
Guided waves (Lamb waves) are extensively used for NDE of plate-like structures. When defects are present, they scatter guided waves, creating mode conversions. This challenge explores how frequency-wavenumber (f-k) analysis can separate defect-scattered waves from incident waves.

### Challenge
Design a signal processing pipeline to detect and localize defects in an aluminum plate using guided wave signals.

Mathematically, for scattered wave measured at position x with time delay t:

where c_g is group velocity and θ is scattering angle.

### Key Equations

**Rayleigh-Lamb equations (symmetric modes):**
```
tan(βh)/tan(αh) = -[4αβk²] / [(k² - β²)²]
```

**Antisymmetric modes:**
```
tan(βh)/tan(αh) = -[(k² - β²)²] / [4αβk²]
```

where:
- α² = (ω/cₗ)² - k²
- β² = (ω/cₛ)² - k²
- h = plate half-thickness
- k = wavenumber

**2D-FFT for f-k analysis:**
```
D(ω, k) = ∫∫ d(x,t) · e^(-j(ωt - kx)) dx dt
```
