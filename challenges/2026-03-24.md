## Derive the Zener Model Dispersion Relation from First Principles

### Problem Context
You've implemented the Zener viscoelastic model in 2D/3D FDTD simulations and validated it against theory. Now, derive the dispersion relation analytically to understand *why* the model behaves as it does.

### Challenge
Derive the phase velocity cₚ(ω) and attenuation α(ω) for the Zener (Standard Linear Solid) model starting from fundamental equations.

### Technical Requirements

**Part 1: Constitutive Relation (45 min)**
The Zener model consists of a spring G∞ in parallel with a Maxwell element (spring Gr in series with dashpot η):

```
       ┌──[G∞]──┐
   σ ──┤        ├── ε
       └──[Gr]─[η]──┘
```

- Write the stress-strain relationship in differential form
- Take the Fourier transform to get G*(ω) = G' + jG''
- Show: G*(ω) = G∞ + Gr·(jωτσ)/(1 + jωτσ)
  where τσ = η/Gr is the relaxation time under constant stress

**Part 2: Dispersion Relation (60 min)**
- Start with the wave equation: ρ·∂²u/∂t² = ∇·σ
- Substitute the complex modulus: G*(ω) = |G*(ω)|·e^(jφ)
- Derive the complex wavenumber: k*(ω) = ω·√(ρ/G*(ω))
- Separate real and imaginary parts:
  - kᵣ(ω) = Re[k*] → phase velocity cₚ(ω) = ω/kᵣ
  - kᵢ(ω) = Im[k*] → attenuation α(ω) = -kᵢ

**Part 3: Low and High Frequency Limits (30 min)**
- Show that as ω → 0: cₚ → √((G∞ + Gr)/ρ) = √(G₀/ρ)
- Show that as ω → ∞: cₚ → √(G∞/ρ)
- Explain the physical meaning of G₀ and G∞

**Part 4: Validation (45 min)**
- Numerically evaluate your derived cₚ(ω) and α(ω)
- Compare with simulation data from `shear_wave_2d_zener.py`
- Plot theory vs simulation on same axes

### Key Equations to Derive

**Complex modulus:**
```
G*(ω) = G∞ + Gr·(jωτσ)/(1 + jωτσ)
      = [G∞ + G₀·(ωτσ)²]/(1 + (ωτσ)²) + j·[(G₀ - G∞)·ωτσ]/(1 + (ωτσ)²)
```
where G₀ = G∞ + Gr is the relaxed modulus

**Phase velocity:**
```
cₚ(ω) = √[2|G*|²/(ρ(|G*| + G'))]
```

**Attenuation:**
```
α(ω) = ω·√[ρ(|G*| - G')/(2|G*|²)]
```

### Hints
- Use |G*|² = G'² + G''²
- The complex wave number is k* = ω√(ρ/G*)
- Phase velocity comes from the real part: cₚ = ω/Re(k*)
- Attenuation comes from the imaginary part: α = -Im(k*)
- Remember: 1/(a + jb) = (a - jb)/(a² + b²)
