# Engineering Challenge: March 29, 2026

## Acoustic Non-Destructive Evaluation (NDE) — Ultrasonic Signal Processing

### Challenge

Design a Python-based signal processing pipeline that can:

1. **Simulate ultrasonic A-scan signals** from a material with embedded defects (delaminations, voids, inclusions)
2. **Apply noise reduction** using wavelet denoising or adaptive filtering
3. **Detect defect echoes** using envelope detection and peak finding
4. **Calculate defect depth** based on time-of-flight and known material velocity

### Background

Acoustic NDE uses high-frequency sound waves to inspect materials without causing damage. In ultrasonic testing:
- A transducer emits pulses into the material
- Echoes return from boundaries and defects
- Time-of-flight (ToF) indicates defect depth: `depth = (velocity × ToF) / 2`

### Requirements

- Simulate a 5MHz ultrasonic pulse in steel (velocity ≈ 5900 m/s)
- Add Gaussian noise and attenuation to simulate real-world conditions
- Implement a defect at 25mm depth with 3mm diameter
- Output: defect location, estimated size, confidence metric

### Bonus

- Generate a B-scan image (multiple A-scans along a scan line)
- Implement synthetic aperture focusing (SAFT) for improved resolution
- Add frequency-domain analysis (FFT) to assess signal quality

### Resources

- `scipy.signal` for filtering and peak detection
- `pywavelets` for wavelet denoising
- `matplotlib` for visualization

**Goal:** Build a foundational ultrasonic NDE simulator that could be extended for real sensor data.
