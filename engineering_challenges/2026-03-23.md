## Extend the 1D Shear Wave Simulator to 2D

### Problem Context
Your existing 1D FDTD shear wave simulator (`shear_wave_1d_simulator.py`) models wave propagation in a single dimension. Real tissue imaging requires 2D (or 3D) wave fields to capture realistic beam patterns, reflections, and dispersion effects.

### Challenge
Extend the simulation framework to 2D and validate against theoretical dispersion predictions.

### Technical Requirements

**Part 1: Parameter Exploration (30 min)**
- Run 1D simulations with varying viscoelastic parameters:
  - Storage modulus G': 1-50 kPa (soft tissue range)
  - Viscosity η: 0.1-5 Pa·s
- Document how dispersion changes with (G', η)

**Part 2: 2D FDTD Implementation (90 min)**
- Extend to 2D grid (e.g., 200×200 points, 0.5 mm spacing)
- Implement shear wave equation in 2D:
  ```
  ρ·∂²u/∂t² = G*·∇²u
  ```
  where G* = G' + jωη (complex modulus)
- Add point source excitation (mimicking acoustic radiation force)

**Part 3: Absorbing Boundaries (45 min)**
- Implement Mur 2nd-order ABC or PML to prevent reflections
- Validate: wave should exit domain without significant reflection

**Part 4: Dispersion Extraction (45 min)**
- Apply 2D FFT (space-time → k-ω domain)
- Extract phase velocity vs frequency: c_p(ω) = ω/k(ω)
- Compare with Kelvin-Voigt theoretical dispersion

### Deliverables
1. Working 2D FDTD code with visualization
2. Dispersion curve plot: simulated vs theoretical
3. Brief write-up of observations (boundary effects, stability limits)

### Hints
- Stability criterion: Δt < Δx / v_max (CFL condition)
- For Kelvin-Voigt: c_p(ω) = √[2(G'²+ω²η²) / (ρ(G'+√(G'²+ω²η²)))]
- Start with small grid, expand once working
