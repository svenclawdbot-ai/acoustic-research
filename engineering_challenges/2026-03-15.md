### Key Equations
**Kelvin-Voigt dispersion:**
```
c(ω) = √(2/ρ) · √(|G*|² / (G' + |G*|))
where |G*| = √(G² + (ωη)²), G = storage modulus
```

**Posterior:**
```
P(G', η | data) ∝ P(data | G', η) · P(G') · P(η)
```
