## 🎯 Today's Challenge: Dispersion Curve Extraction

### Background
You've been working on shear wave elastography and have 1D/2D FDTD simulations running. Now it's time to extract meaningful dispersion curves from wave field data — the key to characterizing viscoelastic materials.

### Tasks

1. **Implement 2D FFT for space-time data**
   - Take your existing wave field output (u(x,t))
   - Apply 2D FFT to get k-ω representation
   - Visualize the k-ω spectrum

2. **Extract phase velocity vs frequency**
   - Identify peak energy at each frequency
   - Convert k → phase velocity: c_p = ω/k
   - Handle aliasing and windowing artifacts

3. **Fit Kelvin-Voigt model**
   - Use extracted dispersion to estimate G' and η
   - Compare with ground truth (if using simulation)
   - Evaluate fitting error

4. **Validate against theory**
   - Plot theoretical Kelvin-Voigt dispersion
   - Overlay extracted data points
   - Compute RMS error

### Deliverable
Python script (`dispersion_extraction_2d.py`) that:
- Loads wave field data
- Computes k-ω transform
- Extracts dispersion curve
- Fits viscoelastic parameters
- Generates validation plots

### Success Criteria
- [ ] Clean k-ω spectrum with visible dispersion branch
- [ ] Extracted phase velocity within 10% of theoretical
- [ ] G' and η estimates within 20% of true values
- [ ] Publication-quality plots

Focus on robustness: your extraction should work even with moderate noise (SNR ~20 dB). Consider using masks or weighted fitting to improve accuracy.
