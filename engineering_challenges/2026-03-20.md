## 🎯 Today's Challenge: Adaptive Array Optimization for Dispersion Measurement

### Background
You've made excellent progress on the inverse problem — your Bayesian MCMC framework is working with 24.7% acceptance rate and you've characterized the G∞–τ_σ degeneracy. Now it's time to apply this knowledge to **experimental design**: designing an ultrasound array that optimally measures dispersion curves for Bayesian parameter recovery.

The key insight: not all frequencies are equally informative for recovering viscoelastic parameters.

### Tasks

1. **Implement Fisher Information Matrix (FIM) computation**
   - Given Zener model parameters θ = [G₀, G∞, τ_σ]
   - Compute ∂c/∂θ for each parameter (analytical derivatives)
   - Build FIM: F_ij = Σ_k (∂c/∂θ_i)(∂c/∂θ_j) / σ_k²
   - Where σ_k is measurement noise at frequency k

2. **D-optimal array design**
   - Use FIM determinant as optimality criterion
   - Optimize element positions for maximum |F|
   - Constrain: minimum element spacing (λ/2 to avoid grating lobes)
   - Constrain: total array aperture (practical size limit)

3. **Compare with uniform array**
   - Baseline: regular spacing N-element array
   - Optimized: D-optimal positions
   - Metric: Cramér-Rao lower bound on parameter variance

4. **Validate with simulated measurements**
   - Generate synthetic dispersion data with realistic noise
   - Run Bayesian inference for both array configurations
   - Compare posterior uncertainty ellipses

### Deliverable
Python script (`adaptive_array_design.py`) that:
- Computes Fisher Information for Zener model
- Optimizes array geometry via gradient descent or genetic algorithm
- Generates comparison plots (FIM determinant, CRLB, beam patterns)
